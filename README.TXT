PROJECT TITLE: First Calculator
PURPOSE OF PROJECT: Making a simple enough calculator for practice purposes
VERSION and DATE: V1.0, 13/01/2019
AUTHORS: Happysausage
USER INSTRUCTIONS: Input a first number, an operator, and a second number
on the command line, separated by a whitespace. Calculator keeps asking for 
inputs after that, untill you close the program.
